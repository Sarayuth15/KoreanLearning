Korean Tenses Summary (Polite Form)

Present Tense (아요/어요)

  • Usage: Describes current actions or states.
  • Formation:
  • If the last vowel of the verb/adjective stem is 아 or 오: add -아요.
  • For other vowels: add -어요.
  • For 하다 verbs: changes to -해요.
  • Examples:
  • 가다 (gada - to go) -> 가요 (gayo)
  • 먹다 (meokda - to eat) -> 먹어요 (meogeoyo)
  • 하다 (hada - to do) -> 해요 (haeyo)
  • 예쁘다 (Yeppeuda - to be pretty) -> 예뻐요 (Yeppeoyo)
  • Past Tense (았어요/었어요/했어요)
  • Usage: Describes actions or states that have already occurred.
  • Formation:
  • If the last vowel is 아 or 오: add -았어요.
  • For other vowels: add -었어요.
  • For 하다 verbs: changes to -했어요.
  • Examples:
  • 가다 (gada - to go) -> 갔어요 (gasseoyo)
  • 먹다 (meokda - to eat) -> 먹었어요 (meogeosseoyo)
  • 하다 (hada - to do) -> 했어요 (haesseoyo)
  • 예쁘다 (Yeppeuda - to be pretty) -> 예뻤어요 (Yeppeosseoyo)
  • Future Tense (-겠다 / -을 것이다 / -를 것이다)
  • Usage: Describes actions or states that will occur.
  • Formation:
  • For strong intention or prediction: add -겠어요.
  • For speculative future:
  • If the verb stem ends in a vowel: add -ㄹ 것이다.
  • If the verb stem ends in a consonant: add -을 것이다.
  • Examples:
  • 가다 (gada - to go) -> 가겠어요 (gageosseoyo) / 갈 것이다 (gal geosida)
  • 먹다 (meokda - to eat) -> 먹겠어요 (meokgeosseoyo) / 먹을 것이다 (meogeul geosida)
  • 하다 (hada - to do) -> 하겠어요 (hageosseoyo) / 할 것이다 (hal geosida)
  • 예쁘다 (Yeppeuda - to be pretty) -> 예쁘겠어요 (Yeppeugeosseoyo) / 예쁠 것이다 (Yeppeul geosida)
  • Summary of Endings:
  • Present: -아요 / -어요
  • Past: -았어요 / -었어요 / -했어요
  • Future: -겠다 / -을 것이다 / -ㄹ 것이다
